Introducing our new safety system

@capitalnerd March 30, 2023

We have launched a new security system that confirms that you really are the person you say you are. To make it harder for hackers to hack you, we have introduced a file system.

You get a unique file as a CodeLively member. This file contains a certain multi-encrypted code. With this code we can confirm that it is really you who you are contacting us. You should keep your file in a safe place to which only you have access. If you contact us now, we may reply that you are attaching your file. Please do this only if the message comes from the email 

codelivelyproject@gmail.com
.

What if I lose it?

If you accidentally delete the file or share it with someone, please contact us with this title: 

Codelively Safety Support - File loss
Please explain in this email what happened. If someone else has access to the file and you want to block it, please send us the file.
